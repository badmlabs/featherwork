# Install drill-steps preview on Android

## Download

**badminton-court-simulator-drill-steps-preview.apk** from:

https://github.com/haritabh17/badminton-court-simulator/releases/tag/drill-steps-preview

## Install

1. Allow installs from unknown sources on your phone.
2. Download the APK and open it, or: `adb install -r badminton-court-simulator-drill-steps-preview.apk`
3. Open **Badminton Court Simulator**.

## Try drill steps

1. Drag players or the shuttle — each move adds a step.
2. Tap **Steps** → name the drill → **Share link** (WhatsApp, etc.).
3. Open the shared link on another phone with this app to load the drill.

Standalone debug preview (no dev server). Package: `com.haritabhgupta.badmintoncourtsimulator`.
